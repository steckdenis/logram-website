{% trans "Le coin du développeur" %}

{% blocktrans %}

Logram prône l'ouverture, et pour pousser ce fait jusqu'à son maximum, tout le développement est ouvert.

Cette page contient les outils de développement en ligne utilisés par les développeurs de Logram, et également à votre disposition. Au fur et à mesure que Logram avancera, les outils seront ajoutés à cette page, qui fera office de «fourre-tout». Ainsi, rien n'est caché, et le reste du site web reste propre, adapté à une communauté grandissante.

{% endblocktrans %}

{% trans "Code source en ligne" %}

{% blocktrans %} Logram utilise le système de gestion des sources Git, qui permet aux développeurs de travailler ensemble, en même temps, et de gérer les changements faits par tout le monde. {% endblocktrans %}

{% blocktrans %} Ce module vous permet de naviger dans la toute dernière version des sources de Logram, et de les récupérer. Ainsi, vous êtes toujours à jour et pouvez poster des rapports de bugs plus pertinents. {% endblocktrans %}

{% blocktrans %} Le dépôt Git de Logram étant hébergé chez Gitorious, vous bénéficiez également de toutes les fonctionnalités de cette forge, comme la possibilité de clôner le dépôt, le modifier directement, puis demander que vos changements soient mergés dans le dépôt officiel. {% endblocktrans %}

{% trans "Morceaux de code" %}

{% blocktrans %} Quelques fois, au hasard de discussions sur IRC ou par mail, ou carrément dans le forum, des développeurs ont besoin de montrer de petits morceaux de code, et éventuellement de les éditer plus tard. Ces morceaux sont temporaires, ils ne vont donc pas dans le dépôt Subversion. {% endblocktrans %}

{% blocktrans %} Ce module vous permet de poster vos propres morceaux de code, supprimés après un certain temps, et de les éditer. Ainsi, vous pouvez partager des idées facilement, avec tout le confort nécessaire (coloration de code, historique des modifications, visualisation des différences). {% endblocktrans %}

{% endblock %}
